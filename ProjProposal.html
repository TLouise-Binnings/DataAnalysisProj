Describe your data, including variables and data types.

The data is the number of male/female people of different race categories that are employed in major Science and Engineering fields along with disability status and type of disability.

All variables are: Science/Engineering Fields, Sex, Race, Number (count) of employees in each category.

The quantities of each type is an integer value, Sex is a string/character, Race is a String and STEM field type is a string.

Have there been any shifts in those proportions between 2019 and 2023?
